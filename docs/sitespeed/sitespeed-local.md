# Sitespeed.io Local

## Running Sitespeed.io locally on Mac

```bash
throttle 3gfast

docker run \
    --name sitespeed-local \
    --network=host \
    --cap-add=NET_ADMIN \
    --privileged \
    --shm-size=2g \
    --rm \
    -v "$(pwd):/sitespeed.io" \
    sitespeedio/sitespeed.io:23.5.0-plus1 \
    -e REPLAY=true \
    -e LATENCY=100 \
    -n 3 \
    -c 3gfast \
    --browser chrome \
    --browsertime.viewPort 1920x1080 \
    --speedIndex \
    --visualElements \
    --video \
    --cpu \
    --axe.enable \
    --chrome.timeline \
    --thirdParty.cpu \
    --sustainable.enable \
    https://example.com

throttle stop
```
